## 📌 Overview

**CreditWise** is a machine learning project designed to predict whether a loan application will be approved based on applicant information and financial attributes.

The project demonstrates a complete ML workflow including:

- Data Cleaning
- Missing Value Handling
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Data Preprocessing
- Model Training & Evaluation
- Loan Approval Prediction
